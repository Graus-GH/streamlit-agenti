# Streamlit Agenti – Ricerca Articoli

Applicazione per la ricerca e selezione di articoli da parte degli agenti, con possibilità di esportazione in PDF ed Excel.

## Funzionalità
- Ricerca full-text su codice, prodotto, categoria, tipologia, provenienza
- Selezione multipla e aggiunta a un paniere
- Rimozione articoli dal paniere
- Esportazione in Excel e PDF

## Setup locale
```bash
git clone https://github.com/TUO-USERNAME/streamlit-agenti.git
cd streamlit-agenti
pip install -r requirements.txt
streamlit run app.py
